# User Stories

## MVP Stories

### User Authentication & Account Management

#### Sign Up
As a new user, I want to create an account so that I can engage with live music events.
#### Select Role
As a new user, I want to select whether my account will be as a concertgoer, performer, or venue
#### Sign In
As a registered user, I want to log in so I can create or view pages for live music events.
#### Password Reset
As a registered user, I want to be able to be able to reset my password if I forget it.

### General

#### Register Venue
As a user, I want to be able to create a new venue that I manage so that I can schedule events.
#### Register Performer
As a user, I want to be able to create a new performer that I manage so I can be connected to venue events.

### Concertgoer Functionality

#### Find Shows By Location
As a concertgoer, I want to be able to search for a city and see a list of upcoming shows in the area.
#### Find Venues By Location
As a concertgoer, I want to be able to search for a city and see venues in the area
#### Find Performers by Location
As a concertgoer, I want to be able to search for a city and see bands local to the area
#### Follow Venues
As a concertgoer, I want to be able to follow venues so I can easily see upcoming events hosted.
#### Follow Performers
As a concertgoer, I want to be able to follow performers so I can easily see upcoming performances.

### Venue Functionality
#### Create Events
As a venue, I want to be able to create new events with date, time, and performers.
#### Update Events
As a venue, I want to be able to change event date/time or performers.
#### Cancel Events
As a venue, I want to be able to cancel a scheduled event.

### Performer Functionality
#### Update Profile
As a performer, I want to update my profile to include member names, genre, and links to socials / streaming / merch.

## Future Development

### General
#### Profile Photos
As a user, I want to be able to upload a photo to represent myself / band / venue.

#### Event Photos / Video
As a user, I want to be able to upload photo or video taken at an event.

### Venues

#### Map
As a venue, I want to be able to include the street address and an embedded map so user's can easily find my venue.
#### Notification and Communication
As a venue, I want to be able to notify followers when an event is created with a custom message
#### Ticketing
As a venue, I want to be able to include a link to an external website where tickets can be purchased for the event.

### Performers

#### Hosting
As a performer, I want to be able to create an event at a venue and edit the lineup.
#### Communication
As a performer, I want to be able to notify my follows with a custom message

### Concertgoers

#### Notifications
As a concertgoer, I want to receive notification of new events at venues I follow or from performers I follow.

#### Attending
As a concertgoer, I want to be able to indicate I intend to attend an event so I can see a list of upcoming events
I'd like to attend and be alerted before they happen.

#### Region Search
As a concertgoer, I want to be able to see events and bands within so many miles of my location.

#### Explore
As a concertgoer, I want to be able to find bands in an area by genre

#### Recommendations
As a concertgoer, I want to be able to be recommended bands or venues I might like based on my past interactions.
